IntelliJ IDEA使用

Posted on 2020-12-03

IntelliJ IDEA使用方法积累（update20201203）

1.基础快捷键

  • 删除当前行：CTRL+Y
  • 复制当前行：CTRL+D
  • 格式化代码：CTRL+ALT+L
  • 单行注释：CTRL+/
  • 多行注释/局部注释：CTRL+SHIFT+/
  • 自动代码生成：ALT+INSERT（生成构造方法和Getter\Setter方法）
  • 移动当前行代码：ALT+SHIFT+上下箭头
  • 5次的for循环：5.fori+回车
  • 修改所有地方的同一个参数：SHIFT+F6+回车
  • 将选中的代码提取到新的方法中 CTRL+ALT+M

简写

  • System.out.println();//简写sout
  • public static void main(String[] args) {//简写psvm
  • for (int j = 0; j < 5; j++) {//简写5.fori
  • for (int i = 5; i > 0; i--) {//简写5.forr

2.IntelliJ IDEA中查看一个类的所有继承关系：包括父类与子类

第一种：

1.找到当前类所在的位置，右键需要查看的类选择：Diagrams->Show Diagram…

2.快捷键：Ctrl+Alt+Shift+U

第二种：

1.定位在需要查看的类当中，点击编辑器最上端的Navigate->Type Hierarchy

2.快捷键：Ctrl+H
